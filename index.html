Active Directory Penetration Test

Problem Statement

Compromise a simulated corporate Active Directory environment to demonstrate escalation paths from a standard user to Domain Admin.

Approach

Began with initial enumeration using Nmap. Captured user hashes via AS-REP Roasting. Escalated privileges utilizing BloodHound to chart a path to the Domain Controller via a misconfigured ACL.

Tools Used

Nmap Impacket BloodHound CrackMapExec

Key Lessons

Gained practical experience mapping Active Directory trusts. Learned the critical importance of restricting service accounts and managing ACL configurations properly.

Advanced SQL Injection Bypass

Problem Statement

Exploit an e-commerce search field protected by a Web Application Firewall (WAF) to extract underlying customer data.

Approach

Used Burp Suite to intercept traffic. Discovered the WAF was filtering standard UNION statements. Bypassed the filter by utilizing Unicode encoding and time-based blind SQL injection techniques.

Tools Used

Burp Suite Pro SQLMap

Key Lessons

Understood how encoding variations can evade signature-based WAFs. Reinforced the necessity of parameterized queries on the backend to prevent SQL injection entirely.
